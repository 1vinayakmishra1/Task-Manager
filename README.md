# Task Manager

A simple, elegant task management web application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Automatic saving to browser's local storage
- Responsive design

## How to Use

1. Open `task-manager.html` in your web browser
2. Type a task in the input field
3. Click "Add Task" or press Enter to add
4. Use the Edit button to modify tasks
5. Use the Remove button to delete tasks
6. Click the checkbox to mark tasks as complete

Your tasks are automatically saved in your browser's local storage and will persist between sessions.

## Technology Stack

- **HTML5** - Structure
- **CSS3** - Styling
- **JavaScript (ES6+)** - Functionality

## Local Storage

This app uses the browser's `localStorage` API to save tasks. Data is stored locally on your device and is not synced across devices or browsers.

## Getting Started

No installation required! Just open the HTML file in any modern web browser.
